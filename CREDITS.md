## Icons
- All UI icons are original inline SVGs, drawn by hand in this codebase.
  See `assets/main.js` -> `svgIcon()` and the recycle bin in `index.html`.

## Sounds
- All UI sounds are synthesized at runtime via the Web Audio API.
  No copyrighted audio is shipped. See `assets/sound.js`.
- The boot chime is an original four-note triad with sparkle harmonics,
  not a copy of any specific operating system startup sound.
